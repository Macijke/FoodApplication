# FoodDeliveryApplication

Aplikacja pozwalająca na zamawianie jedzenia z różnych restauracji.

Wykorzystywane technologie:
   - Express.JS
   - Bootstrap
   - Mongoose
   - Pug


# Wszystkie komendy:

``
npm install pug-cli -g
``

``
pug -w .\public\ -o ./html
``

``
npm install nodemon -g
``

``
nodemon server.js
``

``
npm install -g sass
``

``
sass --watch .\public\style.scss .\css\style.css
``
