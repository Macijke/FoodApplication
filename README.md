# FoodDeliveryApplication

Aplikacja pozwalająca na zamawianie jedzenia z różnych restauracji.

Wykorzystywane technologie:
   - Express.JS
   - Bootstrap
   - Mongoose
   - PUG
   - Express-Validator
   - Cookie-Parser


# Wszystkie komendy:


``
npm install nodemon -g
``

``
nodemon server.js
``

``
npm install -g sass
``

``
sass --watch .\public\style.scss .\public\css\style.css
``
